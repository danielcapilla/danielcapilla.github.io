Balloon World Championship

Balloon World Championship, inspired by Ibai Llanos and Gerard PiquÃ©'s popular tournament, is an exciting twist on the classic "don't let the balloon touch the ground" game. This game involves players hitting a balloon to keep it in the air without it touching the floor. What I used: Phaser (HTML,CSS,JavaScript), Spring Boot (Java), Visual Studio Code and Github.

Zynxle

Zynxle is a multiplayer educational app featuring mini-games designed to help students develop their computational thinking skills while playing and having fun in a group setting. Teachers can easily monitor game statistics and track their students' progress.
In the available mini-game, Restaurant, students have to fulfill incoming orders using example hamburgers, practicing pattern recognition and abstraction. After playing, the results of all student games are saved in a database, allowing teachers to review them anytime.

What I did:I implemented the entire network communication system for the game, utilizing Unity's Relay service to enable multiplayer connectivity. I developed the lobby system, designed the database-driven menu, and contributed to the implementation of part of the mini-game functionality.

What I used: Unity (C#), Visual Studio, Netcode for GameObjects and Github.

You can view the code here: Zynxle

OpenGL

This project focuses on the creation of a 3D scene using OpenGL, demonstrating proficiency in the mathematics behind it, shader programming for lighting, and utilizing the OpenGL API.

What I did:I implemented a 3D scene from scratch using the OpenGL API, following the rasterization graphics pipeline. This project included the development and integration of various types of lighting, such as point lights, directional lights, and spotlights, with distance attenuation functions and penumbra effects for the spotlights. Additionally, I implemented a normal map to enhance the surface detail of objects in the scene.
Furthermore, I developed another scene incorporating two shaders: one for preprocessing and another for postprocessing. These shaders allowed me to add effects such as motion blur, Gaussian blur, Depth of Field, and various convolution masks.

What I used: C++, GLSL, OpenGL, Visual Studio and Github.

Computer Graphics

This project focuses on real-time rendering and physically based shading, featuring the development of a Vulkan renderer and a path tracing implementation in Nori.

What I did: I implemented the rendering passes required for physically based rendering (PBR) in Vulkan, including a depth pre-pass, SSAO, and shadow mapping. I also addressed common shadow mapping artifacts using bias correction and PCF filtering for soft shadows. Additionally, I implemented RTX-based shadows using acceleration structures (BLAS and TLAS) to achieve realistic lighting and reflections. Beyond the real-time renderer, I completed a set of path tracing exercises in Nori, implementing recursive light transport, BSDF sampling, and indirect illumination.

What I used: Vulkan, GLSL, C++ and RenderDoc.

You can view the code here: Vulkan Renderer and Path Tracer
